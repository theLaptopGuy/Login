# Simple Login in Ruby On Rails

* Basic Registration as User , Log In and Log Out .

* Ruby Version 2.6.3 .

* Rails Version 5.2.3 .

* Database SQLite3 .

* Run `bundle update` then `rails server` in Login folder .
